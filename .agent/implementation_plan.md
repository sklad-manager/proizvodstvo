# План реализации: MES-система (Argon Dashboard Edition)

Проект будет базироваться на шаблоне Argon Dashboard Tailwind для обеспечения премиального внешнего вида в светлых тонах.

## Технологический стек
- **Framework**: Next.js 14+ (App Router)
- **Styling**: Tailwind CSS (с использованием кастомного конфига Argon)
- **Icons/Fonts**: Nucleo Icons, Open Sans (из шаблона)
- **Database**: Supabase (PostgreSQL)
- **Deployment**: Local dev -> GitHub -> Vercel

## Интеграция дизайна Argon
1. **Конфигурация**: Перенос `tailwind.config.js` и `postcss.config.js` из шаблона.
2. **Ассеты**: Копирование шрифтов, иконок и изображений в папку `public/`.
3. **Глобальные стили**: Интеграция `styles.css` и специфичных для Argon CSS-файлов в `app/globals.css`.
4. **Компоненты**: Создание React-компонентов для Sidebar, Navbar и Footer на основе HTML-верстки шаблона.

## Структура приложения
1. **Layout**: Общий контейнер с боковым меню и верхней панелью.
2. **Страницы (Вкладки)**:
   - `/` — Dashboard (Обзор производства).
   - `/raw-materials` — Учет сырья.
   - `/products` — Учет готовой продукции.
   - `/schedule` — График работы.
   - `/maintenance` — Ремонт и обслуживание.
   - `/expenses` — Запланированные траты.
   - `/finance` — Финансовый отчет.

## Этапы работы
### Этап 1: Инициализация (Текущий)
- Создание Next.js приложения.
- Настройка Tailwind с конфигом Argon.
- Перенос ассетов в `public/`.
- Создание базового Layout (Sidebar + Navbar).

### Этап 2: База данных и API
- Настройка проекта в Supabase.
- Создание таблиц для сырья и продукции.
- Настройка CRUD API.

### Этап 3: Реализация модулей
- Разработка страниц для каждой из 6 вкладок.
- Интеграция форм ввода и таблиц в стиле Argon.
